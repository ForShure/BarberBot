# 💈 Telegram Bot для записи в Барбершоп

## 📝 Описание
Fullstack Telegram-бот для автоматизации записи клиентов к мастерам.
Проект решает проблему ручной записи, предоставляя клиентам удобный интерфейс для выбора мастера, даты и свободного времени (слотов).

## 🚀 Функционал
- **Клиентская часть:**
  - Регистрация пользователей (/start).
  - Интерактивный календарь (выбор даты на ближайшую неделю).
  - Умная фильтрация слотов (нельзя записаться на занятое время).
  - Просмотр списка мастеров.
- **Админ-панель:**
  - Управление мастерами и записями через Django Admin.
- **Бэкенд:**
  - Асинхронная архитектура (Aiogram 3).
  - ORM для работы с БД (Django).
  - Защита от дублирования записей.

## 🛠 Технологический стек
- **Язык:** Python 3.10+
- **Фреймворк бота:** Aiogram 3.x
- **База данных:** PostgreSQL + Django ORM
- **Инструменты:** Asgiref (sync_to_async adapter)

## 📦 Как запустить
1. Клонируйте репозиторий.
2. Создайте файл `.env` с токеном бота.
3. Включить Docker
4. Запустить контейнеры `docker-compose up -d --build`
5. Выполнить миграции `docker-compose exec web python web/manage.py migrate`
6. Создать супер пользователя `docker-compose exec web python web/manage.py createsuperuser`
